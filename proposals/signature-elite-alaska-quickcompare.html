All four cabin tiers have the same bed setup — two twins that convert to a king. Request king configuration at booking.

All four cabin tiers have the same shower — there is no bathroom upgrade between Veranda and Concierge. The shower concern is the same across all options.

AquaClass spa access is included in fare. Veranda and Concierge guests can purchase spa day passes separately — so spa access isn't exclusive to AquaClass, just bundled.
